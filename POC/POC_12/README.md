# POC-12: Documentation/tutorial usability PoC

This PoC checks whether a non-expert can complete signing flow using docs only.

## Objective

- provide a short quick-start tutorial
- run a self-check from clean steps
- capture and fix friction points

## Run

```bash
./venv/bin/python POC/POC_12/generate_poc12_docs_usability.py
```

## Artifacts

Generated in `POC/POC_12/artifacts/`:
- `poc12_quickstart_tutorial.md`
- `poc12_friction_points.md`
- `poc12_transcript.txt`
